# Git Cheatsheet

### Start a new project

```shell
$ mkdir project_name
$ git init
$ touch readme.md
$ git add readme.md
$ git commit -m "initial commit"
```

### Do some work and thn save it

First, do some work!
Make some changes., put them in the box, label the box.

```shell
$ git status
```
